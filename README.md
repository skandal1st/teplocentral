# teplocentral.org

Сайт муниципального предприятия отопления, водоснабжения и водоотведения с административной панелью.

## Возможности

- Публичный сайт с разделами:
  - Контакты
  - Закупки
  - Информация о товарах и услугах
  - Раскрытие информации
  - Профсоюз
  - Вакансии
- Админ-панель для редактирования контента всех разделов.
- Загрузка файлов через админ-панель и публикация ссылок на них на сайте.
- Дизайн на базе палитры и принципов из брендбука.

## Запуск

```bash
npm install
npm start
```

Сайт будет доступен по адресу `http://localhost:3000`.

## Доступ в админ-панель

По умолчанию:

- Логин: `admin`
- Пароль: `admin123`

Поменять можно через переменные окружения:

- `ADMIN_USER`
- `ADMIN_PASSWORD`
- `SESSION_SECRET`

## Где хранятся данные

- Контент разделов: `data/content.json`
- Список загруженных файлов: `data/files.json`
- Сами загруженные файлы: `uploads/`
